# BilPrisEstimering

BilPrisEstimering er et Python-basert prosjekt som estimerer bilpriser ved hjelp av maskinlæring. Modellen bruker et datasett med bilens egenskaper som merke, modell, år, kjørelengde, drivstofftype og girtype for å forutsi prisen på en bil.

## Teknologier brukt
- **Python**: Hovedprogrammeringsspråk for backend-logikk.
- **Flask**: Web-rammeverk brukt for å lage en REST API som håndterer forespørsler om bilpriser.
- **Scikit-learn**: Brukt til å bygge og trene en Random Forest-modell.
- **Joblib**: Brukt til å lagre og laste modellen.
- **Pandas**: Brukt til databehandling og rensing.
